2. What We Collect 收集內容

✅ Business name, address, industry category, registration ID (public records)

❌ We do NOT collect: personal phone numbers, personal emails, photos, user reviews, social media profiles

3. Business Owner Rights 企業權利

  • Claim: Verify and manage your listing at /claim
  • Update: Add or correct business information after verification
  • Opt-out: Request complete removal within 48 hours at /opt-out

4. AI Usage Policy AI 使用政策

This directory is designed for AI engines. Data may be used for:

  • AI-generated answers about businesses in these regions
  • Training data for language models (government open data terms apply)
  • Search engine indexing and knowledge graph building

5. Security 安全措施

  • No user accounts or authentication data stored
  • Content Security Policy (CSP) headers enabled
  • X-Frame-Options: DENY to prevent clickjacking
  • Rate limiting on API endpoints
  • Regular security audits of data pipeline
  • No third-party trackers or advertising scripts

Data compliance inquiries: GitHub Issues
